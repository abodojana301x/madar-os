🎨 6. Theme System (خفيف جدًا)
🎯 مش engine ضخم… بس values
typedef struct {
    int bg;
    int panel;
    int window;
    int text;
    int accent;
} Theme;

🌗 Light / Dark:
Theme dark = {
    0x1E1E1E,
    0x2A2A2A,
    0x333333,
    0xFFFFFF,
    0x4CAF50
};

🔄 ربط مع Settings:
current_theme = dark;
