1.2 Taskbar
📐 الخصائص:
position: bottom
height: 44px
layout: centered
🎨 العناصر:
Start Button
App Icons (static)
Clock (يمين)
struct Taskbar {
    int x, y;
    int width, height;
};

void render_taskbar() {
    draw_rect(0, SCREEN_H - 44, SCREEN_W, 44, COLOR_PANEL);

    // Start button (center)
    draw_rect(center_x - 100, SCREEN_H - 40, 40, 40, COLOR_BTN);

    // Clock (right)
    draw_text(SCREEN_W - 80, SCREEN_H - 25, "12:45");
}

1.3 Start Menu (Launcher)
📐 الخصائص:
يظهر فوق الـ taskbar
hidden by default
bool start_menu_open = false;

🎨 المحتوى:
Grid apps (ثابتة)
void render_start_menu() {
    if (!start_menu_open) return;

    draw_rect(center_x - 200, SCREEN_H - 300, 400, 250, COLOR_WINDOW);

    draw_text(center_x - 150, SCREEN_H - 260, "Terminal");
    draw_text(center_x - 150, SCREEN_H - 230, "Settings");
}

🖱️ 2. نظام الإدخال (Input System)
🎯 المطلوب فقط:
Mouse click
تحديد هل المستخدم ضغط على Start
void handle_mouse_click(int x, int y) {

    // Start button area
    if (x > center_x - 100 && x < center_x - 60 &&
        y > SCREEN_H - 40) {

        start_menu_open = !start_menu_open;
    }
}

🔄 3. Game Loop (Core Loop)

النظام كله في loop بسيط:

while (1) {

    handle_input();

    render_desktop();
    render_taskbar();
    render_start_menu();

    swap_buffers();
}
