🖥️ 2. Terminal (أهم تطبيق)
🎯 الهدف:

تنفيذ أوامر نصية بسيطة

🧱 المكونات:
Text area (output)
Input line
📐 Structure:
typedef struct {
    AppWindow window;

    char input_buffer[256];
    char output_buffer[1024];

} TerminalApp;

⌨️ إدخال المستخدم:
void terminal_on_key(int key) {

    // إضافة للحقل
    append_char(input_buffer, key);

    if (key == KEY_ENTER) {
        execute_command(input_buffer);
        clear_buffer(input_buffer);
    }
}

⚙️ تنفيذ أوامر بسيطة:
void execute_command(char* cmd) {

    if (strcmp(cmd, "help") == 0) {
        append_output("Commands: help, clear");
    }

    else if (strcmp(cmd, "clear") == 0) {
        clear_output();
    }

    else {
        append_output("Unknown command");
    }
}
